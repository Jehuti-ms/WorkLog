Setup Instructions:
1. Create a new Google Sheet
2. Enable Google Sheets API in Google Cloud Console
3. Create an API key and make your sheet publicly readable
4. Enter your Sheet ID and API Key above
5. Click "Initialize Sheets" to automatically create all required sheets and headers
6. For write operations, deploy the Google Apps Script and enter the Web App URL
